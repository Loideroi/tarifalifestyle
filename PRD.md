## 1. Executive Summary

Tarifalifestyle.com is a multilingual lifestyle and resource website designed to be the definitive guide for expats considering relocating to or currently living in Tarifa, Spain. The website will serve as a curated hub connecting visitors with essential services, local businesses, and the unique kitesurf-centric beach lifestyle that defines this southernmost point of continental Europe.

The platform will feature deep integrations with partner businesses, particularly Tarifa Air Force clothing store via Shopify, alongside practical information for daily expat life including healthcare, education, coworking, and real-time beach/wind conditions.

---

## 2. Problem Statement

### Current Challenges for Tarifa Expats

- **Fragmented Information:** Essential expat resources are scattered across multiple platforms, languages, and outdated websites
- **Language Barriers:** Many local resources are Spanish-only, creating friction for incoming expats
- **Discovery Problem:** Quality local businesses and services are often found only through word-of-mouth
- **Integration Gap:** No single platform connects the lifestyle aspects (kitesurfing, beach culture) with practical living needs (healthcare, schools, coworking)

### Market Opportunity

Tarifa has become an increasingly popular destination for:
- Digital nomads seeking a kitesurf lifestyle
- Families relocating from Northern Europe
- Retirees attracted to the climate and cost of living
- Wind sports enthusiasts making permanent moves

## 4. Target Audience

### Primary Personas

#### 1. "The Prospector" - Pre-Move Research Phase
- **Demographics:** 25-45, professional, EU citizen
- **Needs:** Housing costs, visa info, quality of life, job/remote work viability
- **Pain Points:** Uncertainty, language barriers, finding trustworthy information
- **Content:** Cost of living guides, "A day in the life" content, visa FAQs

#### 2. "The New Arrival" - First 6 Months
- **Demographics:** Recently relocated, setting up life
- **Needs:** NIE process, healthcare registration, finding services, making friends
- **Pain Points:** Bureaucracy, loneliness, not knowing local systems
- **Content:** Step-by-step guides, business directory, community events

#### 3. "The Established Expat" - Living the Life
- **Demographics:** 1+ years in Tarifa, integrated
- **Needs:** Staying current, finding deals, community news
- **Pain Points:** Missing local updates, finding reliable services
- **Content:** News integration, partner offers, event calendar

#### 4. "The Kite Tripper" - Seasonal Visitor
- **Demographics:** Visits 2-4 weeks/year for kitesurfing
- **Needs:** Real-time conditions, gear services, accommodation
- **Pain Points:** Limited local knowledge, wasted days due to conditions
- **Content:** Wind forecasts, kite repair, lessons, gear shops

### Secondary Personas

- **Families with Children:** School and daycare information priority
- **Digital Nomads:** Coworking, internet quality, community focus
- **Retirees:** Healthcare, slower pace content, integration services

## 7. Integrations

### 7.1 Primary Integration: Tarifa Air Force (Shopify)

**Priority:** Critical
**Complexity:** Medium
**Decision Date:** January 28, 2026

#### Integration Strategy (UPDATED)

After evaluating Shopify's API options, we will use a **tiered approach**:

##### Tier 1: Admin API (Server-Side) - PRIMARY
Using Shopify Admin API credentials for server-side product fetching.

**Credentials (ENV VARS - NEVER EXPOSE CLIENT-SIDE):**
```
SHOPIFY_CLIENT_ID=<your-shopify-client-id>
SHOPIFY_SECRET=<your-shopify-secret>
SHOPIFY_STORE_URL=tarifaairforce.com
```

**Implementation:**
- All Shopify calls via Next.js **Server Components** or **API Routes**
- Cache responses aggressively (ISR with 1-hour revalidation)
- Never expose credentials to client/browser
- Use `@shopify/shopify-api` package for Admin API

##### Tier 2: Static Product Data - FALLBACK
If API issues arise, maintain a curated static product catalog:

**Approach:**
- Scrape/export key products from tarifaairforce.com
- Store in CMS (Sanity) or JSON files
- Manual updates monthly or on request
- Deep links directly to product pages on tarifaairforce.com

**Products to Feature (curated selection):**
- 10-15 bestselling items per category (Men/Women)
- Seasonal highlights
- New arrivals (manual updates)

##### Tier 3: Shopify Catalog MCP - FUTURE OPTION
Shopify offers a Catalog MCP for AI agents to search global product catalogs.
- URL: `https://discover.shopifyapps.com/global/mcp`
- Uses same credentials with OAuth token exchange
- Better for product discovery across multiple stores
- Consider for future "shop similar items" features

#### Why This Approach?

| Option | Pros | Cons | Decision |
|--------|------|------|----------|
| Storefront API | Safe for frontend, designed for headless | Requires additional token setup in confusing Shopify UI | Skipped |
| Admin API (server) | Works with existing credentials, full access | Must be server-side only | **PRIMARY** |
| Static/Scrape | Simple, fast, no API dependency | Manual updates needed | **FALLBACK** |
| Catalog MCP | AI-native, global search | Overkill for single store | Future |

#### Features to Implement

1. **Product Catalog Display**
   - Fetch products via Admin API (server-side)
   - Categories: Men, Women, Accessories
   - Filter by tags: beach, summer, kite, casual
   - Cache with ISR (revalidate: 3600)

2. **Product Cards**
   - Image galleries (CDN-hosted Shopify images)
   - Price display in EUR
   - Size/variant indicators
   - **"Shop Now" CTA** → Deep link to tarifaairforce.com product page

3. **Curated Collections (CMS-managed)**
   - "Beach Day Essentials"
   - "Kite Session Gear"
   - "Tarifa Sunset Look"
   - "New Arrivals"
   - Collections defined in Sanity CMS, products fetched from Shopify

4. **Technical Implementation**
   ```typescript
   // lib/shopify/admin-client.ts
   import { shopifyApi, LATEST_API_VERSION } from '@shopify/shopify-api';

   const shopify = shopifyApi({
     apiKey: process.env.SHOPIFY_CLIENT_ID!,
     apiSecretKey: process.env.SHOPIFY_SECRET!,
     scopes: ['read_products'],
     hostName: 'tarifalifestyle.com',
     apiVersion: LATEST_API_VERSION,
     isEmbeddedApp: false,
   });

   // Use in Server Components or API routes ONLY
   ```

5. **Affiliate/Referral Tracking**
   - UTM parameters on all outbound links: `?utm_source=tarifalifestyle&utm_medium=website&utm_campaign=shop`
   - Click tracking via GA4 events
   - Monthly referral reports for partner
